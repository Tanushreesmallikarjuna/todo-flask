# 📝 Flask TODO App

A simple and clean TODO web application built using Flask.  
This app allows users to manage daily tasks efficiently with a minimal UI.

---

## 🚀 Features
- ➕ Add new tasks  
- ❌ Delete tasks  
- 📋 View all tasks  
- ⚡ Fast and lightweight  

---

## 🛠️ Tech Stack
- Python  
- Flask  
- HTML  
- CSS
